### Run locally

Would like to run this project locally? Open terminal and follow these steps:
1. Clone the repo
```
git clone hhttps://github.com/RamintaVIL/serlokas
```
2. Install NPM packages
```
npm i
```
  or
```
npm install
```
3. Run the server
```
npm run dev
```

__There is no tests for this project.__
